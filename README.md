# Invitacion de aniversario

Proyecto web listo para subir a Netlify o conectar con GitHub Pages.

## Archivos principales

- `index.html`: estructura de la invitacion.
- `style.css`: diseno visual.
- `script.js`: datos editables, contador, musica y enlaces.
- `video/recuerdo-aniversario.mp4`: video principal.
- `music/romantica-suave.wav`: musica de fondo.
- `img/`: carpeta para agregar fotos si deseas ampliar la galeria.

## Cambiar datos

Abre `script.js` y edita el bloque `INVITATION`:

- `names`: nombres principales.
- `dateISO`: fecha real para el contador.
- `dateText` y `timeText`: texto visible.
- `place`: lugar visible.
- `mapUrl`: enlace de Google Maps.
- `whatsappNumber`: numero con codigo de pais, sin signos. Ejemplo: `50212345678`.

## Subir a Netlify

1. Entra a Netlify.
2. Ve a Add new site -> Deploy manually.
3. Arrastra esta carpeta completa o el archivo ZIP.
4. Copia el enlace que te da Netlify.
5. Convierte ese enlace en QR.
